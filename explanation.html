How to Erase

Erasure is done here using a special syntax in the "text to erase" box to the left of the button.
At a very basic level, you can type in a literal word that you want to erase in the box and every instance of that word will be erased. The patterns below can be combined in any way, and the gallery below provides some examples of combined patterns. Caution should be taken when using these patterns with large samples of text, as the processing time for tagging and NLP can be quite large and slow down and freeze web browsers.
In this guide, I highlighted just a few features of the underlying technology for erasure that I think are the most applicable to the domain of literature that this tool is intended for. For a more comprehensive overview of the syntax, pleace check out the reference page in the comprimise.js repository, which is what powers this project.

Parts of Speech (Tags)

To erase tags, type in a pound sign followed by the appropriate tag, i.e. #Noun to erase all nouns.

Suffix, Prefixes, and Infixes

Using an underscore character servers as a wildcard for any amount of characters, so _ed erases all words ending in -ed, _in_ erases any words with 'in' in the middle, and so on

This or That

You can define a list of things that could possibly be matched by enclosing them in parentheses and seperating them by pipes. For example, (run|dash|sprint) will match a group of fast words, or (very|really|#adverb) will erase text according to a selection of writing advice from established authors.

Anything, Everything, and Nothing

A general wildcard expression is just a period (.), and matches to any kind of word. #noun . . erases any noun followed by two words of any kind.
A greedy wildcard matches to at least one and maybe more of a given token in a row. #noun+ erases any amount of nouns in a row, such as "John, Mark, Luke".
A potential marker optionally matches a word, but doesn't require it. An example is #person [#adverb]? runs, which would erase both the sentence "John runs" and "John swiftly runs".

Regular Expressions

Regular expression syntax works as well on the text here, although slower than many other erasing strategies. All that's needed is enclose the regular expression in /.../ forward slashes.

Other Features

Inversion

Pressing invert is a neat feature that flips the erased and normal text around. I have found it just as useful for creating cool erasure work.

Uploads

The upload button creates a large box where you can paste in text, and a url box for text file urls. Many sites that host text files do not allow them to be cross-referenced or downloaded by this tool, so pasting in text is recommended.
Note that all pasted text will keep it's original format on the web page.
